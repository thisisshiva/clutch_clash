## Notes
- Banner text sits near center so YouTube TV/mobile crops don’t cut the wordmark.
- YouTube standard uploads stay landscape (~3 min); YouTube Shorts + Instagram Reels use a 60s vertical cut.
- Publish auto-picks `brand/thumbnails/{track}.png` when available; otherwise falls back to the template.
